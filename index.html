Rpi-gpio

What is Rpi-gpio?

  • Rpi-gpio is a simple application that makes interfacing with the Raspberry Pi GPIO easier. 
  • You will be able to control the I/O pins of the Raspberry Pi by using a few simple commands without the need to write any code.
  • Reading, writing and "blinking" (turning off and on according to certain intervals) are the I/O operations that are currently supported. Features such as reading the analog value from a sensor and conditional "if" statements will be added in future releases.

Current version: v1.1
